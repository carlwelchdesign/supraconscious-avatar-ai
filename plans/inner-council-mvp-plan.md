# Inner Council MVP Plan

## Summary
Inner Avatar becomes a Maria-grounded spiritual reflection system, not an AI Maria persona and not a generic journaling assistant. The first release centers on a bounded Inner Council session: the user enters through a Threshold prompt, writes or speaks, receives concise council perspectives, answers one Integrator question, and saves one embodied micro-shift.

## Product Frame
- The guide is a reflective/spiritual presence inspired by Maria Olon Tsaroucha's teachings.
- The guide is not Maria, does not speak for Maria, does not channel, and does not claim supernatural certainty.
- The product is reflective journaling, not therapy, diagnosis, crisis care, medical care, or professional advice.
- Responses are reflective possibilities grounded in user text and approved source material.

## MVP Flow
1. Threshold: a daily prompt or entry question invites the user into the session.
2. Opening: the user writes or speaks what is present.
3. Observer: the system names the core tension in grounded language.
4. Council: fixed archetype agents respond through bounded lenses.
5. Integrator: the Supraconscious Guide asks exactly one clarifying question.
6. Embodiment Gate: the user saves one small shift to carry forward.
7. Memory: recurring signals update only when pattern memory is enabled.

## Council Roles
- Protector: risk, fear, protection, avoidance, and emotional safety.
- Conditioned Self: learned roles, inherited scripts, habits, and social conditioning.
- Visionary: future potential, expansion, values, and what wants to emerge.
- Truth Self: contradiction, clarity, reality-testing, and agency-preserving honesty.
- Supraconscious Guide: synthesizes the council into one question and one grounded integration step.

## Non-Goals
- No AI Maria persona.
- No public sharing, marketplace, or creator tools.
- No philosopher/domain-expert agents in the MVP.
- No dynamic archetype generation.
- No long open-ended chat interface.
- No advanced reports or streak mechanics.

## Acceptance Criteria
- A first-time user can complete Threshold to Embodiment Gate in under three minutes.
- Every council voice is concise and bounded to its role.
- The Integrator returns exactly one question.
- High-risk safety inputs bypass council confrontation and receive grounding support.
- Pattern memory respects user consent.
- The current journal flow remains available behind feature flags during rollout.
